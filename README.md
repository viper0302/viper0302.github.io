# Udacity Memory Game Project

This is the classic Memory Game/Concentration game.

## How to play this game

* Go to this link [Memory Game](https://viper0302.github.io)
* Clicking on the cards will turn them over
* Your goal is to match the cards in as minimum of moves as possible
* Good luck!!

### Things To Know

* I made this as part of the Udacity Front End Web Developer nanodegree.

 



### How to Run

* Download or clone this repository
* open in your favorite text editor
* open the index.html in your favorite browser
* and enjoy!!
